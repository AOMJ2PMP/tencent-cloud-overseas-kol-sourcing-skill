# Creator Evaluation

## 先做硬性判断

| Gate | 通过标准 |
| --- | --- |
| Platform | 活跃于指定平台且主要内容形态匹配 |
| Language | 近期主要内容使用指定语言 |
| Geography | 达到 strict base 或 audience geo 规则 |
| Content fit | 能自然承载产品 demo/教程/体验 |
| Budget | low/base 报价在可用预算附近 |
| Timing | 有现实机会在目标日期前交付 |
| Safety | 无明确排除项、有效排他或重大交付风险 |

## 六维评分

每项 0-5，必须写一句证据理由。

### 产品与内容自然度（25%）

- 5：近期持续覆盖同类问题，产品可成为内容中的必要工具。
- 3：赛道相符，但需要较强 brief 才能自然。
- 1：只因粉丝多而相关。

### 受众与目标市场（20%）

- 5：有可靠受众地域/职业/兴趣证据。
- 3：creator base 或内容语境吻合，但 audience analytics 缺失。
- 1：只有宽泛地域或语言猜测。

### 近期表现质量（15%）

- 5：同格式内容中位表现稳定，相关题材表现良好。
- 3：表现波动或样本较少。
- 1：依赖旧爆款、Shorts 虚高或近期不活跃。

### 商业合作可行性（15%）

- 5：有商务入口、近期同类 sponsor、交付形式成熟。
- 3：可联系且接过合作，但同类证据有限。
- 1：无商务入口、极少接推广或明显只做自有项目。

### 预算匹配（15%）

- 5：high 估算仍在分配预算内。
- 4：base 在预算内。
- 3：low 在预算内，可通过 scope 谈判。
- 1：low 已超过预算。

### 品牌安全与交付风险（10%）

- 5：无明显风险，内容质量和披露习惯稳定。
- 3：有可管理的事实核查、档期或品牌语调风险。
- 1：存在明确冲突、误导性内容惯例或高交付风险。

## 可合作性证据

强信号：

- 公开 business email、经纪页或 media kit
- 过去 12 个月有明确披露的科技/B2B/开发者赞助
- 有 dedicated review、tutorial、integration 或探访案例
- 能在简介/网站看到 partnership 入口
- 赞助内容仍保持合理表现

弱信号：

- 只在很早以前接过一次赞助
- 只有 affiliate links
- 内容很相关但没有任何商业入口

风险信号：

- 近期已有直接竞品合作且疑似排他
- 需要无法提供的产品权限、地点或设备
- 内容周期远长于 campaign 时间
- 只接受高制作 dedicated，而预算只够 integration
- 经常删除、延迟或不披露商业内容

竞品合作不是自动淘汰。需要判断合作时间、scope 和排他条款。

## Reach tier

头部应按市场和垂类定义：

- `Market head`：在目标国家/语言市场中有广泛影响力。
- `Vertical head`：在开发者、数据库、AI、支付等垂类中有高权威。
- `Mid-tier`：规模适中、内容匹配和合作概率通常更高。
- `Micro authority`：粉丝较少，但对采购者/开发者影响强。

不要把“100 万粉”当作所有国家和垂类的统一头部门槛。

## 推荐层级

- `Primary`：硬门槛通过，>=70 分，预算 base 可行，证据中高。
- `Secondary`：硬门槛通过，但报价或受众证据需确认。
- `Wildcard`：有战略价值但预算/地域/合作概率较弱，不能占用主名单名额。
- `Reject`：写清淘汰原因。
